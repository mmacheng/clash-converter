## 🚀 如何使用？

1. 打开 [在线工具](https://mmacheng.github.io/clash-converter/index.html)
2. 粘贴节点链接（每行一个）：
3. 点击 **生成 Clash 配置**
4. 下载 `clash_config.yaml` 或扫码导入手机
